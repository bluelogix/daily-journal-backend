# Endpoints
| For | Method  | Endpoints |
| ------------- | ------------- | ------------- |
| Register  | POST  | /api/register|
| Login  |   POST | /api/login |
| Create |   POST | /api/journals |
| Update  |  PUT | /api/journals/:id |
| Delete  |  DELETE | /api/journals/:id|
